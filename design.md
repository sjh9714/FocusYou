# design.md — Focus You

> macOS 메뉴바 집중력 앱. 차단 + 타이머를 하나로.

---

## 1. 앱 컨셉

**한줄 설명:** "타이머를 시작하면 방해가 사라진다."

**앱 이름:** Focus You — "당신에게 집중하세요"
**App Store 부제:** Block Distractions & Focus Timer

### 왜 이 앱인가?

현실: 집중하려면 1Focus(차단)과 Be Focused(뽀모도로)를 따로 설치해야 한다.
두 앱은 연동이 안 되므로, 타이머가 끝나도 차단이 안 풀리고, 차단을 켜도 타이머가 안 돈다.

Focus You의 답: **타이머 시작 → 차단 자동 활성, 타이머 종료 → 차단 자동 해제.**
이 한 가지가 핵심이다. 나머지는 부가가치다.

### 3초 엘리베이터 피치

```
"시작 버튼 하나로 — 사이트 차단, 앱 차단, 타이머가 동시에 시작.
 끝나면 전부 자동 해제. 그게 Focus You예요."
```

### 디자인 원칙

1. **출시된 제품 > 완벽한 계획.** 기능을 빼는 용기.
2. **집중을 방해하는 집중 앱이 되지 말 것.** 알림 최소화, 회고 강제 금지.
3. **프로그레시브 디스클로저.** 첫 실행은 버튼 하나. 쓸수록 기능이 나타남.
4. **매 버전이 완성품.** v0.1도 그 자체로 쓸 만한 앱이어야 한다.

## 2. 버전 로드맵

> 핵심 규칙: 각 버전은 **그 상태로 배포해도 사용 가능한 완성품**이다.

### 전체 흐름

```
v0.1  "차단되는 타이머"        ← 내가 매일 쓸 수 있는 최소 제품
v0.3  "뽀모도로 차단 앱"       ← 지인에게 보여줄 수 있는 수준
v0.5  "예쁜 집중 앱"           ← 베타 테스트 배포
v1.0  "출시할 수 있는 앱"      ← 첫 공개 배포 (Gumroad 등)
v1.x  "사용자가 원하는 기능"   ← 피드백 기반 반복
v2.0  "Pro + App Store"        ← 수익화 시작
v3.0  "플랫폼"                 ← AI + iOS + 확장
```

### v0.1 — "차단되는 타이머" (1.5주)

> 목표: 타이머 시작 → 사이트/앱 차단, 타이머 종료 → 차단 해제.
> 이것만 되면 내가 매일 쓸 수 있다.

#### 사용자 시나리오
```
1. 메뉴바 아이콘 클릭
2. "25분" 선택
3. ▶ 집중 시작
4. → facebook.com, twitter.com 차단됨
5. → 슬랙, 카카오톡 강제 종료됨
6. 25분 후 알림 → 차단 자동 해제
```

#### 구현 범위

**메뉴바 앱**
- macOS 14+ MenuBarExtra (.window 스타일)
- Dock 숨김 (LSUIElement = YES)
- 메뉴바 아이콘: SF Symbol "shield.fill" (비활성: 회색, 활성: 빨강)
- 팝오버: 340 × 400pt

**자유 타이머**
- 프리셋: 25분, 45분, 1시간, 2시간
- 커스텀: 1분 ~ 4시간 (드래그 또는 입력)
- 상태: idle → running → completed
- 중앙에 큰 숫자: "23:45" (카운트다운)
- 취소: 확인 다이얼로그 (실수 방지)
- 완료: 시스템 알림 + 사운드

**웹사이트 차단**
- /etc/hosts 파일 수정 (127.0.0.1 리다이렉트)
- 마커: `# === Focus You BEGIN ===` ~ `# === Focus You END ===`
- www 서브도메인 자동 추가
- AuthorizationServices sudo 권한
- DNS 캐시 flush (dscacheutil + mDNSResponder)
- 시작 시 백업, 해제 시 마커 구간 제거

**앱 차단**
- NSWorkspace.didLaunchApplicationNotification 감시
- 차단 앱 실행 → terminate (실패 시 forceTerminate)
- 이미 실행 중인 앱도 종료
- Bundle Identifier 매칭
- 차단 시 알림: "{앱이름}이 차단되었습니다"

**차단 목록 관리**
- 웹사이트: URL 직접 입력 (자동 정리: https:// 제거, 소문자)
- 앱: /Applications 스캔, 체크박스 선택
- 프리셋 카테고리 버튼: SNS, 뉴스, 동영상, 게임
- 데이터 저장: SwiftData

**핵심 연동 (이 앱의 존재 이유)**
- 타이머 시작 → 차단 즉시 활성화
- 타이머 종료/취소 → 차단 즉시 해제
- 앱 크래시 대비: LaunchAgent로 차단 해제 안전장치

```
│          [ 23:45 ]               │  ← 큰 숫자
```

### v0.3 — "뽀모도로 차단 앱" (+1주)

> 목표: 1Focus + Be Focused를 대체할 수 있는 수준.
> 뽀모도로 사이클이 차단과 연동된다.

#### v0.1 대비 추가되는 것

**뽀모도로 엔진**
- 기본 사이클: 집중 25분 → 휴식 5분 → (×4) → 긴 휴식 15분
- 모든 시간 커스터마이즈 가능
- 세션 카운터: "3/4 뽀모도로"
- 자동 전환 옵션 (집중 끝 → 자동으로 휴식 시작)
- Overflow 모드: 25분 끝나도 "계속하기" 선택 가능, 추가 시간 기록

**차단 ↔ 뽀모도로 연동**
- 집중 사이클 → 차단 활성
- 휴식 사이클 → 차단 해제
- Overflow 중 → 차단 유지

**파이차트 비주얼 타이머**
- 원형 파이차트로 남은 시간 시각화
- 드래그로 시간 설정 가능
- 상태별 색상: 집중 🔴, 휴식 🟢, Overflow 🟡
- 중앙 숫자 + 하단 뽀모도로 카운터

**메뉴바 개선**
- 남은 시간 텍스트 표시: "🛡 18:32"
- 아이콘 색상: 집중(빨강), 휴식(초록)

**모드 선택**
- 팝오버 상단에 [🍅 뽀모도로] [⏱ 자유] 토글
- 모드에 따라 하단 UI 변경 (뽀모도로: 사이클 설정 / 자유: 시간 설정)

#### 팝오버 UI (v0.3)
```
┌──────────────────────────────────┐
│                                  │
│      ╭──────────────╮            │
│      │   🔴         │            │  ← 파이차트
│      │   18:32      │            │
│      │              │            │
│      ╰──────────────╯            │
│       3/4 뽀모도로                │
│                                  │
│   [🍅 뽀모도로]  [⏱ 자유]        │
│                                  │
│   집중 25분 / 휴식 5분 / 긴휴식 15분│
│                                  │
│   ┌────────────────────────┐    │
│   │    ▶  집중 시작          │    │
│   └────────────────────────┘    │
│                                  │
│   🛡 차단: SNS, 뉴스 외 3개      │
│   [ 🌐 차단 목록 관리 ]          │
│                                  │
└──────────────────────────────────┘
```

#### 완료 기준
- [x] 뽀모도로 사이클 자동 전환 (집중→휴식→집중→...→긴휴식)
- [x] Overflow 선택지 (계속하기/휴식하기) 동작
- [x] 집중 중 차단, 휴식 중 해제 정확 동작
- [x] 파이차트 타이머 렌더링 + 드래그로 시간 설정
- [x] 뽀모도로/자유 모드 전환
- [x] 뽀모도로 시간 커스터마이즈

### v0.5 — "예쁜 집중 앱" (+1주)

> 목표: 지인에게 "이거 써봐" 하고 보여줄 수 있는 수준.
> 테마로 예쁘고, 프로필로 편리하다.

#### v0.3 대비 추가되는 것

**테마 시스템**
- 10개 기본 테마 (미니멀 5개, 따뜻한 3개, 차가운 2개)
- 테마 = Primary Color + Secondary Color + Background Style
- 테마 변경 시 모든 UI 즉시 반영: 파이차트, 메뉴바 아이콘, 버튼
- 테마 갤러리: 2열 그리드 + 프리뷰
- ThemeManager가 모든 색상의 단일 진실 원천

**차단 프로필**
- 프로필 = 차단 목록 + 타이머 모드/시간 묶음
- 기본 3개:
  - 🔴 "딥 워크" — SNS+뉴스+메일 차단, 자유 타이머 2시간
  - 🟢 "공부 모드" — SNS+동영상+게임 차단, 뽀모도로 25/5
  - 🔵 "글쓰기" — 전체 인터넷 차단, 자유 타이머 1시간
- 커스텀 프로필 CRUD
- 팝오버에서 프로필 탭 → 원클릭 시작
- 프로필별 아이콘(SF Symbol) + 색상

**기본 통계 (오늘만)**
- 오늘 집중한 총 시간
- 오늘 완료한 뽀모도로 수
- 팝오버 하단에 한 줄: "⏱ 2h 35m  🍅 ×6"

**세션 기록**
- 모든 완료된 세션을 SwiftData에 저장
- 시작/종료 시간, 모드, 프로필, 실제 집중 시간, Overflow 여부

### v1.0 — 첫 공개 출시 (+2주)

> 목표: 돈을 내지 않아도 매일 쓰고 싶은 무료 앱.
> Gumroad / 개인 사이트로 첫 공개 배포.

#### v0.5 대비 추가되는 것

**Flowmodoro**
- 세 번째 모드: [🍅 뽀모도로] [⏱ 자유] [🌊 Flow]
- 고정 시간 없이 자유롭게 집중, 멈추고 싶을 때 정지
- 집중 시간의 1/5을 휴식으로 자동 부여:
  - 25분 집중 → 5분 휴식
  - 50분 집중 → 10분 휴식
  - 90분 집중 → 18분 휴식
- 카운트 업(⬆️) 표시: "▲ 47:23"
- 예상 휴식 시간 실시간 표시: "멈추면 9분 24초 휴식"
- 파이차트: 블루 🔵, 채워지는 방향

**주간 통계**
- 7일 바 차트 (Swift Charts)
- 일별 집중 시간, 뽀모도로/Flowmodoro/자유 모드 구분
- 오늘의 요약 확장: 총 시간 + 뽀모도로 수 + Flowmodoro 수

**스트릭**
- 매일 최소 1세션 완료 시 스트릭 유지
- 🔥 표시: "🔥 7일 연속!"
- 최장 스트릭 기록

**온보딩**
- 첫 실행 시 3스텝 온보딩:
  1. "Focus You에 오신 걸 환영합니다" + 핵심 가치 설명
  2. "차단할 사이트/앱을 선택하세요" (카테고리 프리셋 원클릭)
  3. "준비 완료! 첫 세션을 시작해보세요"
- 30초 안에 첫 세션 시작이 목표

**완성도 마무리**
- 취소 방지 강화: 30초 쿨다운
- 완료 애니메이션: 체크마크 + 컨페티
- 스트릭 업 애니메이션: 🔥 바운스
- 다크모드/라이트모드 전체 테스트
- 접근성: VoiceOver 기본 지원

## 3. 기술 아키텍처

### 서비스 구조

```
BlockingCoordinator (actor, singleton)
├── WebsiteBlocker (protocol)
│   ├── HostsFileBlocker (v1, sudo, 직접 배포)
│   └── NetworkExtensionBlocker (v2, App Store)
├── AppBlocker (NSWorkspace Notification 기반)
└── 상태: idle / blocking / error

TimerCoordinator (actor)
├── FreeTimer (카운트다운)
├── PomodoroEngine (사이클 상태 머신 + Overflow)
└── FlowmodoroEngine (카운트업 + 비례 휴식)

ThemeManager (@Observable, singleton)
├── 현재 테마
├── 테마 카탈로그 (JSON)
└── 모든 색상의 단일 진실 원천

AppState (@Observable)
├── 현재 상태 (idle / focusing / break / overflow)
├── 현재 프로필
├── 오늘의 통계 (캐시)
└── 프로그레시브 디스클로저 레벨
```

### 데이터 모델

```swift
// v1.0부터 iCloud 호환 타입만 사용 (향후 iOS 동기화 대비)

@Model class BlockProfile {

    var icon: String                // SF Symbol name

}

@Model class FocusSession {
```

### 성능 원칙

```
경쟁 앱 최악의 리뷰: "consume too many resources on Mac"

Focus You 성능 규칙:
1. 메뉴바 유휴 시 CPU 0%
2. 타이머 갱신: 팝오버 열림 → 1초, 닫힘 → 60초
3. 앱 차단: NSWorkspace Notification (polling 절대 금지)
4. SwiftData @Query는 필요한 View에서만
5. 사운드: AVAudioEngine 하나, 미사용 시 해제
6. 위젯: Timeline Provider 15분 간격
7. 딤 처리: CALayer opacity (거의 무자원)
```

### 알림/인터럽트 원칙

```
경쟁 앱 최악의 리뷰: "focus app that interrupts my focus"

1. 집중 중 절대 팝업/알림으로 방해하지 않음
2. 세션 완료 알림은 시스템 알림 1회만 (반복 X)
3. 회고 프롬프트에 "건너뛰기" 항상 표시
4. "리뷰 남겨주세요" 팝업은 절대 집중 중에 표시 안 함
5. 모든 알림은 설정에서 개별 ON/OFF
```
