# Focus You Privacy Policy

Last updated: 2026-05-06

Focus You is a macOS focus timer that helps users block distracting websites and apps during user-started focus sessions. This policy explains what data the app handles and how it is stored.

## Data Collection

Focus You does not collect, sell, or share personal data with the developer or any third party.

The app does not include remote analytics, advertising SDKs, crash reporting services, tracking pixels, or automatic upload of diagnostic data.

## Data Stored Locally

Focus You stores app data locally on the user's Mac, including focus sessions, profiles, blocked website/app settings, schedules, badges, preferences, and diagnostics state needed for app recovery.

This data is stored on device using Apple system storage APIs and is not uploaded by Focus You.

## User-Selected Files

Focus You can create local backups, diagnostic bundles, export files, and recovery previews only when the user explicitly chooses those actions in the app.

When the user selects a folder or file, Focus You uses that selected location only for the requested action. The app does not automatically scan, upload, delete, or modify unrelated files.

Diagnostic bundles are generated locally and are designed to redact sensitive details such as home directory paths, domains, email-like strings, profile names, and app-name candidates where possible. The user decides whether to share any generated diagnostic bundle.

## Calendar Access

Calendar access is optional. If enabled by the user, Focus You can write completed focus sessions to Apple Calendar. Calendar data is not uploaded by Focus You.

## Purchases

Focus You uses Apple's StoreKit APIs for Pro purchases, subscriptions, and purchase restoration. Payment processing and Apple ID purchase history are handled by Apple. Focus You does not receive or store payment card details.

## Network Extension

The Mac App Store build uses Apple's Network Extension/System Extension APIs to block websites selected by the user during active focus sessions. The App Store build does not modify `/etc/hosts`, does not install a privileged helper, and does not use sudo.

# Focus You 개인정보 처리방침

최종 업데이트: 2026-05-06

Focus You는 사용자가 직접 시작한 집중 세션 동안 방해 웹사이트와 앱을 차단하도록 돕는 macOS 집중 타이머입니다. 이 문서는 앱이 어떤 데이터를 다루고 어디에 저장하는지 설명합니다.

## 데이터 수집

Focus You는 개인 데이터를 개발자 또는 제3자에게 수집, 판매, 공유하지 않습니다.

앱에는 원격 분석, 광고 SDK, 원격 크래시 리포팅, 추적 픽셀, 진단 데이터 자동 업로드 기능이 포함되어 있지 않습니다.

## 로컬 저장 데이터

Focus You는 집중 세션, 프로필, 차단 웹사이트/앱 설정, 스케줄, 배지, 환경설정, 앱 복구에 필요한 진단 상태를 사용자의 Mac에 로컬로 저장합니다.

이 데이터는 Apple 시스템 저장 API를 통해 기기에 저장되며 Focus You가 업로드하지 않습니다.

## 사용자가 선택한 파일과 폴더

Focus You는 사용자가 앱에서 명시적으로 선택한 경우에만 로컬 백업, 진단 번들, 내보내기 파일, 복구 미리보기를 생성합니다.

사용자가 폴더나 파일을 선택하면 Focus You는 해당 선택 위치를 요청된 작업에만 사용합니다. 앱은 관련 없는 파일을 자동으로 스캔, 업로드, 삭제, 수정하지 않습니다.

진단 번들은 로컬에서 생성되며 홈 디렉터리 경로, 도메인, 이메일처럼 보이는 문자열, 프로필명, 앱 이름 후보 등 민감할 수 있는 정보를 가능한 범위에서 제거하도록 설계되어 있습니다. 생성된 진단 번들의 공유 여부는 사용자가 직접 결정합니다.

## 캘린더 접근

캘린더 접근은 선택 사항입니다. 사용자가 Apple Calendar 동기화를 켠 경우 Focus You는 완료된 집중 세션을 Apple Calendar에 기록할 수 있습니다. 캘린더 데이터는 Focus You가 업로드하지 않습니다.

## 구입

Focus You는 Pro 구입, 구독, 구입 복원에 Apple StoreKit API를 사용합니다. 결제 처리와 Apple ID 구입 내역은 Apple이 처리합니다. Focus You는 결제 카드 정보를 받거나 저장하지 않습니다.

## Network Extension

Mac App Store 빌드는 Apple Network Extension/System Extension API를 사용해 활성 집중 세션 동안 사용자가 선택한 웹사이트를 차단합니다. App Store 빌드는 `/etc/hosts`를 수정하지 않고, privileged helper를 설치하지 않으며, sudo를 사용하지 않습니다.
